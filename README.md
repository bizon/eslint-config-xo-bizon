## Install

```
$ yarn add --dev eslint-config-xo-bizon
```

### Example usage with XO

```json
{
  "name": "my-awesome-project",
  "devDependencies": {
    "xo": "*",
    "eslint-config-xo-bizon": "*"
  },
  "xo": {
    "extends": "xo-bizon",
    "semicolon": false,
    "space": 2
  }
}
```
